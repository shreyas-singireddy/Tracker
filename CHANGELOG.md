# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-07-01

### Added
- Initial release of FitOS — Offline AI Fitness Operating System
- Modular architecture with core, models, repositories, services, and UI layers
- SQLite-based persistence with migration system
- Workout tracking with session lifecycle management
- Nutrition tracking with deterministic macro calculations
- Habit tracking with consistency scoring
- Recovery monitoring with readiness assessment
- Rule-based AI Coach (no ML, no external APIs)
- Analytics engine with fitness scores and progress tracking
- Streamlit web UI with 9 pages
- Offline-first design with zero network dependencies
- AGPLv3 license
